# 网页内容抓取工具

一个基于 Tkinter 的简单桌面工具，输入网页 URL 后，抓取页面中的所有超链接（`<a>` 标签），并将链接文本与绝对地址列表化显示。

## 功能特性
- 输入 URL，一键抓取页面内所有 `a[href]`。
- 自动将相对链接转换为绝对链接（基于输入 URL 的 `scheme://netloc`）。
- 滚动文本区域显示结果，便于查看较多条目。
- 提供“清空”按钮快速重置输入与结果。

## 运行环境
- Python 3.8 及以上版本。
- 依赖库：`requests`、`beautifulsoup4`（BS4）。
- 操作系统：Windows（其他平台也可运行，只要支持 Tkinter）。

## 安装依赖
在终端/PowerShell 中执行：

```bash
pip install requests beautifulsoup4
```

## 使用方法
1. 在终端/PowerShell 进入项目目录：
   ```bash
   cd c:\Users\Administrator\Desktop\发布软件仓库项目\采集网址
   ```
2. 运行程序：
   ```bash
   python 采集网址.py
   ```
3. 在界面中：
   - 输入目标网页的完整 URL（如 `https://example.com/`）。
   - 点击“获取内容”。
   - 在下方滚动区域查看“标题: 绝对链接”列表。
   - 点击“清空”可重置输入与结果。

## 工作原理概述
- 使用 `requests.get` 发送 HTTP 请求，并设置常见 `User-Agent` 与 10 秒超时。
- 通过 `response.raise_for_status()` 保证非 2xx 状态抛出异常。
- 使用 `BeautifulSoup(html, 'html.parser')` 解析页面。
- 遍历所有 `<a>` 标签并读取 `href` 与文本：
  - 文本通过 `get_text(strip=True)` 获取。
  - 若 `href` 为相对地址，使用 `urllib.parse.urlparse` 解析输入 URL 的基础部分（`scheme` 与 `netloc`），组装为绝对地址。
- 将标题与链接以列表形式输出到滚动文本框。

## 注意事项与限制
- 不支持需要 JavaScript 渲染的内容（本工具仅解析静态 HTML）。
- 相对链接的组装基于简单的 `scheme://netloc` 拼接，未处理 `<base>` 标签、重定向等复杂场景，个别页面可能存在不准确。
- 请遵守目标网站的 `robots.txt` 与使用条款，合理设置访问频率，避免给网站造成压力。
- 可能出现编码问题、证书校验问题、或反爬措施导致的请求失败。
- 本工具仅用于学习与测试，请勿用于违法或违规用途。

## 常见问题排查
- 请求超时/连接异常：
  - 检查网络与 URL 是否有效。
  - 适当增大 `timeout` 或加入重试逻辑。
- 证书错误：
  - 可临时使用 `verify=False`（不推荐），或正确配置证书。
- 页面需要登录或有反爬：
  - 本工具不支持此类场景。
- 抓取结果为空：
  - 页面可能没有 `<a>` 标签，或内容由 JS 渲染。
- 相对链接解析不正确：
  - 页面可能使用 `<base>` 标签，目前未处理；可扩展逻辑以支持该标签。
